1. 初始化磁盘：
- 使用**parted <Disk name> print**查看磁盘信息：
    ```Shell
    #磁盘还未分区
    [ray@CentOSTest ~]$ sudo parted /dev/sdc print
    错误: /dev/sdc: unrecognised disk label
    Model: Msft Virtual Disk (scsi)                                           
    Disk /dev/sdc: 53.7GB
    Sector size (logical/physical): 512B/4096B
    Partition Table: unknown
    Disk Flags: 
    ```

- 使用**parted来分区**，使用gpt分区表：
    ```Shell
    [ray@CentOSTest ~]$ sudo parted /dev/sdc
    GNU Parted 3.2
    使用 /dev/sdc
    Welcome to GNU Parted! Type 'help' to view a list of commands.
    (parted) mklabel gpt
    (parted) quit
    ```

- 再次使用**parted <Disk name> print**查看磁盘信息：
    ```Shell
    #可以看到磁盘已经有GPT分区格式了，显示table为gpt
    [ray@CentOSTest ~]$ sudo parted /dev/sdc print
    Model: Msft Virtual Disk (scsi)
    Disk /dev/sdc: 53.7GB
    Sector size (logical/physical): 512B/4096B
    Partition Table: gpt
    Disk Flags: 

    Number  Start  End  Size  File system  Name  标志
    ```

- 使用**gdisk来分区**

    ```Shell
    [ray@CentOSTest ~]$ sudo gdisk /dev/sdc
    GPT fdisk (gdisk) version 1.0.3

    Partition table scan:
    MBR: protective
    BSD: not present
    APM: not present
    GPT: present

    Found valid GPT with protective MBR; using GPT.

    Command (? for help): n
    Partition number (1-128, default 1): 1
    First sector (34-104857566, default = 2048) or {+-}size{KMGTP}: 
    Last sector (2048-104857566, default = 104857566) or {+-}size{KMGTP}: +18.8G
    Current type is 'Linux filesystem'
    Hex code or GUID (L to show codes, Enter = 8300): 
    Changed type of partition to 'Linux filesystem'
    #保存并退出
    Command (? for help): w

    Final checks complete. About to write GPT data. THIS WILL OVERWRITE EXISTING
    PARTITIONS!!

    Do you want to proceed? (Y/N): Y
    OK; writing new GUID partition table (GPT) to /dev/sdc.
    The operation has completed successfully.
    ```
- 通过**partprobe更新**分区表信息(核心更新)

    ```Shell
    [ray@CentOSTest ~]$ sudo partprobe
    ```

- 通过**lsblk**看磁盘分区状态
    ```Shell
    [ray@CentOSTest ~]$ lsblk /dev/sdc
    NAME   MAJ:MIN RM SIZE RO TYPE MOUNTPOINT
    sdc      8:32   0  16G  0 disk 
    └─sdc1   8:33   0  16G  0 part 
    ```

- 查看 **/proc/partitions**看看**核心**是否能看到分区记录
    ```Shell
    # 核心就能看到分区记录了
    [ray@CentOSTest ~]$ cat /proc/partitions 
    major minor  #blocks  name

    8       32   16777216 sdc
    8       33   16775168 sdc1
    8        0   52428800 sda
    8        1          9 sda1
    8       16  134217728 sdb
    8       17     512000 sdb1
    8       18  133192687 sdb2
    8       30       4096 sdb14
    8       31     506880 sdb15
    ```
